Sneaks-API 是一个基于 Node.js 的球鞋数据接口，封装了四大转售平台的抓取逻辑。你只需传入关键词或款式编号，即可获取商品信息、图片链接、各平台价格映射表。

$ npm install sneaks-api
StockX Goat Stadium FlightClub
const SneaksAPI = require('sneaks-api');
const sneaks = new SneaksAPI();

// 搜索球鞋
sneaks.getProducts("Yeezy", 10, cb);

// 查询价格（尺码映射）
sneaks.getProductPrices("FY2903", cb);

// 热门商品
sneaks.getMostPopular(10, cb);

API
三个核心方法

Sneaks-API 对外暴露三个方法，涵盖搜索 → 详情 → 热门的完整流程。

METHOD 01
getProducts

搜索球鞋

传入关键词（品牌/型号/配色），返回匹配的商品数组，每条包含基础信息和各平台链接。

sneaks.getProducts("Yeezy Cinder", 10, callback)
METHOD 02
getProductPrices

价格详情

传入 Style ID，返回完整商品信息，包含各平台、各尺码的价格映射表（price map）。

sneaks.getProductPrices("FY2903", callback)
METHOD 03
getMostPopular

热门商品

传入数量上限，返回 StockX 当前热门商品列表，由 StockX 官方 curation 算法驱动。

sneaks.getMostPopular(10, callback)

实现
Scraper 层架构

每个平台有独立的 Scraper，遵循统一的接口规范：getLink() 补充链接，getPrices() 补充价格。

  stadiumGoodsScraper.getLink();
  goatScraper.getLink();
}

  // 2. 再并发 5 个请求获取各平台价格
  // 3. 计数器 == 5 时才返回
  stockXScraper.getPrices();
  goatScraper.getPrices();
}
